````markdown
# Flutter Tower Challenge

Flutter Tower Challenge adalah mini-game multiplayer realtime yang dibangun menggunakan **Flutter**, **Flame Engine**, dan **Firebase Realtime Database**.  
Game ini mensimulasikan tantangan kompetitif antar tim untuk menyelesaikan tower dengan nilai target tertentu.

---

# 🚀 Tech Stack

Project ini menggunakan teknologi berikut:

- Flutter 3.38.7
- Flame Engine
- Firebase Core
- Firebase Authentication
- Firebase Realtime Database

Dependencies utama:

- flame: ^1.30.1
- flame_forge2d: ^0.19.0+4
- firebase_core: ^4.5.0
- firebase_auth: ^6.2.0
- firebase_database: ^12.1.4
- uuid: ^4.5.3
- collection: ^1.19.1

# ⚙️ Setup Project

Ikuti langkah-langkah berikut untuk menjalankan project di perangkat Anda.

## 1. Clone Repository

Jalankan perintah berikut:

```bash
git clone https://github.com/fahmi54321/flutter_tower_challenge.git
cd flutter_tower_challenge
```
````

---

## 2. Gunakan Versi Flutter

Project ini menggunakan **Flutter versi 3.38.7**.

Cek versi Flutter dengan perintah berikut:

```bash
flutter --version
```

Jika menggunakan **FVM**, jalankan:

```bash
fvm use 3.38.7
```

---

## 3. Install Dependencies

Jalankan perintah berikut untuk mengunduh semua dependency yang dibutuhkan oleh project:

```bash
flutter pub get
```

---

## 4. Tambahkan Firebase Configuration

Download file **google-services.json** dari link berikut:

[https://drive.google.com/file/d/1jRMU9vNzHBQYGTEL7l3p9uRKMW6NYH_d/view?usp=sharing](https://drive.google.com/file/d/1jRMU9vNzHBQYGTEL7l3p9uRKMW6NYH_d/view?usp=sharing)

Setelah didownload, letakkan file tersebut pada folder berikut:

```
android/app/google-services.json
```

File ini diperlukan agar aplikasi dapat terhubung dengan **Firebase**.

---

## 5. Jalankan Aplikasi

Pastikan emulator Android atau device sudah terhubung.

Kemudian jalankan perintah berikut:

```bash
flutter run
```
